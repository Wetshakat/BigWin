1. Overview

BigWin is a next-generation digital lottery platform built on Starknet, designed to deliver fairness, transparency, and seamless user participation.

The platform enables players to enter automated lottery rounds where participants compete for a shared prize pool, with smart contracts governing the entire lifecycle of each round.

Core Mechanics

Each round accommodates up to 50 players

New rounds open automatically as users join

Rounds conclude when:

Player capacity is reached, or

A predefined time limit expires

Winner selection is executed transparently on-chain
